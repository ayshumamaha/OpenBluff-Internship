Project Title

Computational Fluid Dynamics (CFD) Analysis of Water Flow Through a 90° Pipe Elbow Using ANSYS Fluent

Project Overview

This project presents a Computational Fluid Dynamics (CFD) analysis of water flowing through a 90° pipe elbow using ANSYS Fluent 2026 R1. The objective of the study is to investigate the flow characteristics, pressure distribution, wall shear stress, and velocity variations within the pipe under steady-state turbulent flow conditions.

The project demonstrates the complete CFD workflow, including geometry creation, mesh generation, material assignment, boundary condition specification, solver configuration, numerical simulation, and post-processing of results.

Project Specifications
Working Fluid: Water
Pipe Configuration: 90° Pipe Elbow
Analysis Type: Steady-State CFD
Solver: Pressure-Based (Implicit)
Turbulence Model: SST k–ω
Inlet Condition: Velocity Inlet (2 m/s)
Outlet Condition: Pressure Outlet (0 Pa Gauge Pressure)
Wall Condition: Stationary No-Slip Wall
Mesh Type: Tetrahedral
Number of Nodes: 284,704
Number of Elements: 267,995
Number of Iterations: 300
Project Objectives
Develop a three-dimensional model of a 90° pipe elbow.
Generate a high-quality computational mesh.
Simulate water flow using ANSYS Fluent.
Analyse velocity and pressure distribution.
Evaluate wall shear stress along the pipe walls.
Visualize the flow using contour plots, vector plots, and pathlines.
Interpret the hydraulic behaviour of the pipe elbow.
